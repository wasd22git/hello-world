# ソフトウェア工学 2026

### Github Pages の使い方

Github Pages は Github のレポジトリのファイルをウェブページに変換して公開することができるようになる機能である。Jekyll を用いることでmarkdownファイルを自動でhtmlに変換させることができる。  
手順としては、まず Github の設定から Jekyll をインストールする。そして、フォルダを作成し、その中で Jekyll でサイトに必要なものを作成する。最後に、マークダウンファイルの内容を変更し、マージすれば完了する。
